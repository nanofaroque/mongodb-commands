# mongodb-commands
Helpful commands to work with mongodb.

### To start mongobd
``` mongod ```

### To use mongo shell
First start the mongodb server and then command 
``` mongo ```
Use help from mongo shell to see all the commands 

### To show the dbs
``` show dbs```

### To use or create a db
``` use dbName ```

### To insert Data
``` db.collectionNames.insert(yourdatajson) ```
